One-time setup

iOS Widget

Rashun at a glance

Your widget follows the rings and colours selected on your Mac automatically.

  1. 1

    Install Scriptable

    Scriptable securely hosts the widget on iOS.

    Open App Store
  2. 2

    Add the Rashun script

    Download the script, then share or open it with Scriptable. You can also copy it and create a script named “Rashun Widget”.

    Download script
  3. 3

    Copy your secure setup code

    This short-lived code pairs Scriptable with this Mac. It contains no AI-provider credentials.

  4. 4

    Run once in Scriptable

    Open “Rashun Widget”, tap Run, then choose Set up from clipboard. Rashun will test the connection and save it securely.

  5. 5

    Add it to your Home or Lock Screen

    Long-press the Home or Lock Screen → Add Widget → Scriptable, then select “Rashun Widget”. Small widgets mirror your selected menu-bar rings.

    To choose a specific ring, set its widget parameter to Source:metric-id, for example Codex:codex-pro-weekly. Leave it blank to use your first selected menu-bar ring. For multiple Lock Screen rings, add the circular Scriptable widget more than once and give each copy a different parameter.

Rashun requests a refresh every 15 minutes, but iOS chooses the actual time based on battery, usage, and other system conditions. The displayed timestamp is the newest reading received from your Mac.
